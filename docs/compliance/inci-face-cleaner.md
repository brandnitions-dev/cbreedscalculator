# INCI Label Draft — Face Cleaner (Tallow Exfoliating Scrub)

> Rinse-off product with physical exfoliant phase (C)  
> Base: Tallow + Jojoba + Beeswax (optional) + Carriers (A) + Actives (B) + Exfoliant (C) + EO

---

## Default Formula Breakdown

| Phase | % Range | Description |
|---|---|---|
| Tallow | ~50–58% | Adjusted down when C phase increases |
| Jojoba | ~10–11% | Sebum mimic |
| Beeswax | 0–8% | Optional structural hold |
| Carrier A | ~12–13% | Carrier oils |
| Active B | ~5–6% | Therapeutic botanicals |
| **Exfoliant C** | **5–15%** | Physical polish |
| Essential Oils | 1–2% | Scent + therapy |
| Vitamin E | 0.4% | Antioxidant |

---

## Example — Sugar + Oat Face Scrub

### English INCI

```
INGREDIENTS: Adeps Bovis (Beef Tallow), Sucrose (Sugar)*, Simmondsia Chinensis 
Seed Oil (Jojoba), Cera Alba (Beeswax), Avena Sativa Kernel Flour (Oat Flour)*, 
Rosa Canina Fruit Oil, Calendula Officinalis Flower Extract, 
Tocopherol, Lavandula Angustifolia Oil.

May contain: Linalool, Geraniol, Limonene.

*Physical exfoliant — dissolves during use.
```

## Usage Instructions (for label)

### English
```
Usage: Apply a small amount to damp skin. Massage in gentle circular 
motions for 30–60 seconds. Rinse thoroughly with warm water. 
Use 2–3 times per week. Avoid eye area.
For external use only. Discontinue if irritation occurs.
```

### Albanian
```
Përdorimi: Aplikoni një sasi të vogël në lëkurë të lagur. Masazhoni me 
lëvizje rrethore të buta për 30–60 sekonda. Shpëlajeni mirë me ujë 
të ngrohtë. Përdorni 2–3 herë në javë. Shmangni zonën e syve.
Vetëm për përdorim të jashtëm. Ndërprisni përdorimin nëse shfaqet 
irritim.
```

## Safety Warnings (for label)

```
⚠️ Contains physical exfoliant particles.
⚠️ Not suitable for broken or sunburned skin.
⚠️ Patch test before first use.
```
