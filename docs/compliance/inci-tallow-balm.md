# INCI Label Draft — Tallow Skin Balm

> Ingredients listed in descending order of concentration (weight %)  
> Based on default formula: 58% Tallow, 11% Jojoba, 8% Beeswax, 13% Carrier A, 6% Active B, 1–2% EO, 0.4% Vitamin E

---

## Face Balm (1% EO)

### English INCI

```
INGREDIENTS: Adeps Bovis (Beef Tallow), Simmondsia Chinensis Seed Oil (Jojoba), 
Cera Alba (Beeswax), [Carrier Oil A — see table below], [Active Oil B — see table below], 
Tocopherol (Vitamin E), [Essential Oil — see table below].
```

### Example — Lavender + Rosehip Face Balm

```
INGREDIENTS: Adeps Bovis, Simmondsia Chinensis Seed Oil, Cera Alba, 
Rosa Canina Fruit Oil, Helianthus Annuus Seed Oil, Calendula Officinalis Flower Extract, 
Tocopherol, Lavandula Angustifolia Oil, Citrus Sinensis Peel Oil Expressed.

May contain: Linalool, Limonene, Geraniol, Citral.
```

## Label Requirements

| Element | Requirement |
|---|---|
| Product name | Trade name + descriptor (e.g., "MOSSKYN Tallow Face Balm") |
| INCI list | `INGREDIENTS:` followed by INCI names, descending % |
| Allergens | `May contain:` followed by EU 26 allergens present |
| Net weight | In grams or ml |
| PAO | Open-jar symbol with months (e.g., `12M`) |
| Batch number | `Lot:` or `Batch:` |
| Responsible Person | Name + EU address |
| Country of origin | "Made in [Country]" |
| Warnings | As applicable (e.g., "For external use only") |
