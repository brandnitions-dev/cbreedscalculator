# Product Information File (PIF) Template

> Per EU Regulation EC 1223/2009, Article 11  
> One PIF per product — kept by the Responsible Person and available to competent authorities on request

---

## PIF Checklist — Per Product

Create a **physical folder + digital copy** for each product containing:

### Section A — Product Description

- [ ] Product trade name
- [ ] Product category (leave-on / rinse-off)
- [ ] Intended use and target area (face, body, lips, hands)
- [ ] Target consumer (age group, skin type if applicable)
- [ ] Form factor (jar, tin, bottle, bar)
- [ ] Net weight / volume
- [ ] Shelf life / PAO (Period After Opening)
- [ ] Photo of finished product with label

### Section B — Qualitative & Quantitative Formula

- [ ] Complete formula with INCI names
- [ ] Percentage of each ingredient (weight/weight)
- [ ] Function of each ingredient (emollient, fragrance, preservative, etc.)
- [ ] Supplier name + batch number for each raw material
- [ ] Formula version number and date

### Section C — Raw Material Documentation

For each ingredient:

- [ ] Certificate of Analysis (CoA) — purity, composition, heavy metals
- [ ] Material Safety Data Sheet (MSDS / SDS)
- [ ] Allergen declaration (especially for essential oils)
- [ ] IFRA compliance certificate (for fragrance materials)
- [ ] Organic / natural certification (if claimed on label)

### Section D — Manufacturing

- [ ] Reference to GMP manual (ISO 22716)
- [ ] Production process description (step-by-step)
- [ ] Batch record for each production run
- [ ] Equipment list and cleaning procedures

### Section E — Safety Assessment

- [ ] **Safety assessment report** signed by a qualified assessor
  - Toxicological profile of each ingredient
  - Exposure assessment (frequency, area, amount)
  - Microbiological quality (challenge test or risk justification for anhydrous)
  - Stability data (at minimum: 3 months accelerated or 12 months real-time)
  - Impurity assessment (heavy metals, pesticides if applicable)
  - Overall safety conclusion

> ⚠️ The safety assessor must have a university diploma in pharmacy, toxicology, medicine, or equivalent (per EC 1223/2009 Art. 10(2))

### Section F — Stability & Testing

- [ ] Stability test results (temperature cycling, shelf life)
- [ ] Microbiological test results (or justification for waiver if anhydrous, pH > 10)
- [ ] Compatibility testing (product ↔ packaging)
- [ ] pH measurement (soap: must be 8–10 after cure)

### Section G — Labeling

- [ ] Label artwork (front + back)
- [ ] INCI ingredient list (see INCI label files)
- [ ] Allergen callouts (EU 26 fragrance allergens)
- [ ] Warnings and usage instructions
- [ ] PAO symbol or expiry date
- [ ] Batch number location
- [ ] Responsible Person name + address
- [ ] Country of origin
- [ ] Net contents

### Section H — Claims & Evidence

- [ ] Any marketing claims (e.g., "moisturizing", "soothing")
- [ ] Supporting evidence for each claim
- [ ] No medicinal claims (not "treats eczema" — use "helps soothe dry skin")

### Section I — Adverse Effects

- [ ] Log of any reported adverse effects
- [ ] Serious adverse effect reports sent to competent authority
- [ ] Corrective actions taken

## CPNP Notification

Before placing any product on the EU market:

1. Register on [CPNP](https://ec.europa.eu/growth/sectors/cosmetics/cpnp_en)
2. Notify each product with:
   - Product category
   - Product name
   - Responsible Person details
   - Country of origin
   - Frame formulation (ingredient categories and concentration ranges)
   - Label photo
   - Original labeling
3. Update notification if formula changes significantly
