## Workflow

```
1. Write GMP manual (this folder)          ← self-declaration, no inspector needed
2. Compile PIF per product                 ← formula, batch records, CoAs, GMP, label
3. Draft INCI labels                       ← highest % first, allergen callouts
4. Safety assessment (by qualified assessor) ← required before market
5. Microbiological + stability testing     ← lab results go into PIF
6. Register Responsible Person (EU)        ← CPNP notification
```

## Regulatory Notes

- **EU Regulation EC 1223/2009** — all cosmetic products placed on the EU market
- **Responsible Person**: must be established in the EU
- **CPNP**: Cosmetic Products Notification Portal — notify before placing on market
- **Safety Assessment**: must be done by a person with a university degree in pharmacy, toxicology, medicine, or similar
- **GMP Standard**: ISO 22716:2007 (cosmetics GMP) — self-declaration is sufficient for micro-producers
