# EU 26 Fragrance Allergens — Reference

> EU Cosmetics Regulation EC 1223/2009, Annex III  
> Must be declared on label when present above:  
> - **0.001%** (10 ppm) in leave-on products  
> - **0.01%** (100 ppm) in rinse-off products

## How to Use

1. Check which EOs are in your formula
2. Look up their allergens in the table above
3. Calculate the allergen concentration in your final product:
   - `EO % in product × allergen % in EO = allergen % in product`
4. If above threshold (0.001% leave-on / 0.01% rinse-off), list under `May contain:`
5. Order allergens alphabetically

### Example Calculation

Lavender at 1% in a leave-on balm:
- Linalool in lavender ≈ 30%
- Linalool in product = 1% × 30% = 0.30% → **above 0.001%** → must declare

---

## Label Format

```
May contain: Citral, Citronellol, Geraniol, Limonene, Linalool.
```

Alphabetical order. Use INCI names only (not common names).
